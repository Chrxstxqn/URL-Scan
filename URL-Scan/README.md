

# Tool info
This tool is a simple but easy to use Malicious URL Scanner written in Python with the ipqualityscore API.

# Installation
* apt update && apt upgrade
* pkg install python
* pkg install git
* pkg install toilet
* pip install requests

# Usage
* cd URL-Scan
* chmod +x url-scan.py
* python url-scan.py
